# Система управления рецептами

Приложение для создания рецептов, масштабирования порций и составления списка покупок.

## Установка

```bash
git clone https://github.com/haz2007/hw2.git
cd hw2
pip install -r requirements.txt
```

## Запуск тестов

```bash
pytest
```
